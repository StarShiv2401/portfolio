I developed the Tech Times web application to provide a simple and focused platform for publishing and reading technology-related articles. The app allows users to easily create, submit, and view posts in a clean and organized feed, making it convenient to share and discover tech news. It is designed to prioritize content clarity, ensuring an uncluttered and distraction-free reading experience. Each post displays only the title and the content, keeping the platform minimalistic and user-friendly.

The application is built using JavaScript, Node.js, EJS, and MongoDB. Node.js and Express handle the backend logic and routing, EJS manages the dynamic templates, and MongoDB securely stores all user posts. Through building this project, I strengthened my skills in full-stack development, server-side rendering, and working with NoSQL databases. The project is currently hosted locally for development and demonstration purposes, with plans to deploy it online in the future.

Technologies used

  • JavaScript
  • Node.js
  • EJS
  • MongoDB
